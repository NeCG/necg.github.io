site
====

installation
------------

1. `sudo apt-get install ruby ruby-dev`
2. `sudo gem install bundle`
3. `git clone git@github.com:necg/site.git`
4. `cd site`
5. `bundle install`
6. `jekyll build`
7. Done. Built files are placed in _site directory (warning: has to be served from serverroot**/~necg**)
